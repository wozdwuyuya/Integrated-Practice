# 智能健康监测系统 - 完整模板

## 一、系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    智能健康监测系统                           │
├─────────────────────────────────────────────────────────────┤
│  传感器层    │  MAX30102  │  MPU6050  │  KY-039  │  SW-420   │
│             │  (心率血氧) │ (六轴IMU) │ (心率)   │ (震动)    │
├─────────────────────────────────────────────────────────────┤
│  处理层     │        健康监测主程序 (health_monitor_main)     │
│             │  ┌─────────┬─────────┬─────────┬─────────┐   │
│             │  │跌倒检测  │心率分析  │呼吸引导  │健康提醒  │   │
│             │  └─────────┴─────────┴─────────┴─────────┘   │
├─────────────────────────────────────────────────────────────┤
│  输出层     │  SSD1306   │  RGB灯   │  蜂鸣器  │  震动马达  │
│             │  (OLED)    │ (状态)   │ (报警)   │ (触觉)    │
├─────────────────────────────────────────────────────────────┤
│  通信层     │           星闪SLE无线通信 (sle_comm)            │
│             │              ↓                                │
│             │         手机端APP                              │
└─────────────────────────────────────────────────────────────┘
```

- `breath_start` - 启动呼吸引导

- `mute` - 静音
- `confirm` - 确认报警

## 五、功能说明

### 1. 心率监测
- 使用MAX30102采集心率血氧
- 阈值：心率>100BPM 或 <60BPM 为异常
- 血氧<90% 为异常

### 2. 跌倒检测
- 融合MPU6050加速度/陀螺仪 + SW-420震动数据
- 判断条件：
  - 加速度 > 2.5g（冲击检测）
  - 倾斜角度 > 45°（姿态变化）
  - 陀螺仪 > 50°/s（快速旋转）
- 输出置信度 0-100%

### 3. 呼吸引导
- 4-7-8呼吸法：吸气4秒 → 屏息7秒 → 呼气8秒
- OLED显示进度条和倒计时

### 4. 异常提醒
- RGB灯闪烁
- 蜂鸣器鸣叫
- 震动马达振动
- OLED文字提示

### 5. 无线通信
- 使用星闪SLE协议
- JSON格式数据传输
- 支持双向通信

## 六、使用方法

### 初始化
```c
#include "health_monitor_main.h"

int main(void) {
    // 初始化系统
    health_monitor_init();
    
    // 主循环
    while(1) {
        health_monitor_loop();
    }
}
```

### 按键功能
- 按键1：确认/取消报警
- 按键2：启动/停止呼吸引导
- 按键3：静音
